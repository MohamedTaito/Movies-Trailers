Project1 - Movie-Trailer-Website
The Movie Trailer Website project uses python to generate a list of movie tiles,
along with its respective box art imagery and movie trailer.

Overview
entertainment_center.py creates a list of movies from movies.json,
and saves the generated web page to Movies_Trailers.html.

How to use the file
     1-	Open entertainment_center.py 
     2-	Choose run from run list 
     3-	The web site will open directly 

To make your own movies

     1-	Make sure you have python installed in your pc
     2-	Open movies_json and change the info of movies with your own 
     3-	Run the entertainment_center.py like I made before
